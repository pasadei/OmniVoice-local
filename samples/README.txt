Place voice sample pairs here.

Each sample consists of an audio file and a text transcript with matching filenames:

  john-doe.wav      + john-doe.txt
  female-narrator.mp3 + female-narrator.txt

Supported audio formats: wav, mp3, flac, ogg, opus, m4a, aac

The .txt file should contain the exact transcript of the audio.
If the .txt is missing, OmniVoice will auto-transcribe using Whisper (slower).
